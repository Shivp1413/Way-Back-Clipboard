This folder contains 2 exe builds for way back monitor configuration and monitor. 
# 🚀 Quick Start (Recommended)

Prebuilt executables are included inside the `exe_builds` folder.

## Available Executables

| File | Purpose |
|------|----------|
| `wbc_monitor.exe` | Starts clipboard monitoring |
| `wbc_config.exe` | Opens configuration and management menu |

---

## Start Monitoring Clipboard

Run:

```text
exe_builds/wbc_monitor.exe
```

This will:

- Monitor copied text
- Monitor copied images
- Store clipboard history securely
- Track clipboard changes in real time

---

## Open Configuration Panel

Run:

```text
exe_builds/wbc_config.exe
```

From here you can:

- Add keyword replacements
- Manage target applications
- Toggle image saving
- Toggle text saving
- Test focus detection
- Configure clipboard behavior
